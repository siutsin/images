# jsoncrack Docker Image

Docker image for [JSON Crack](https://github.com/AykutSarac/jsoncrack.com) – a data visualisation tool for JSON, YAML, XML, CSV, and more.

## Why

This image is built periodically from the upstream repository to provide an up-to-date containerised version served via nginx.

## Usage

```bash
docker run --rm -p 8080:8080 jsoncrack
```
